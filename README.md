# CORTEX
### Neural Design Engine Powered by TRIBE v2

Cortex is a high-performance neuro-linter designed to detect and eliminate cognitive overload in user interfaces. By utilizing Meta Fundament AI Research team's **TRIBE v2** foundation model, Cortex acts as a digital twin of the human brain, predicting neurological stress responses to visual stimuli in real-time.

---

## The Vision
The original idea for Cortex was a CLI-based tool for developers to lint their code for accessibility. However, we realized that cognitive friction is a deeply visual and biological phenomenon. We pivoted to a high-fidelity visual dashboard to provide a more intuitive and impactful experience. This allows developers to see exactly how their designs affect the human brain through interactive 3D heatmaps and real-time neural telemetry.

---

## Core Technology: TRIBE v2
Meta's TRIBE v2 (TRImodal Brain Encoder) is a breakthrough in in-silico neuroscience. It is a transformer-based architecture that integrates features from text, video, and audio to predict brain activity across the human cortical surface.

### Key Breakthroughs
*   **Neural Prediction:** Predicts how approximately 70,000 brain voxels respond to any digital input.
*   **Zero-Shot Capability:** Understands brain responses for tasks and designs it has never seen before.
*   **Biologically Grounded:** Built on over 1,100 hours of fMRI data from hundreds of subjects.

---

## How it Works
Cortex creates a seamless bridge between raw code and biological limits.

1.  **Capture:** The engine takes a screenshot of your UI and captures the underlying JSX or Tailwind code.
2.  **Simulate:** The data is sent to our Python backend where TRIBE v2 simulates the neural response of a human user.
3.  **Analyze:** We map activation levels across the Visual Cortex and Prefrontal regions to calculate a Friction Score.
4.  **Refactor:** If the friction is too high, an AI agent automatically rewrites the code into a clean, low-load component.

## Technical Stack
Cortex is built with an aerospace-grade architecture for maximum performance and elegance.

*   **Frontend:** Next.js 16 with Turbopack and Tailwind CSS v4.
*   **3D Engine:** Spline and WebGL for interactive neural heatmaps.
*   **Middleware:** Node.js and Express API Gateway.
*   **Neural Core:** Python FastAPI hosting the TRIBE v2 model.
*   **AI Agent:** Google Gemini 2.5 Flash for intelligent code reconstruction.
